Практическое задание 5. Построение схемы организации рабочего места при подготовке поверхностей к монтажу каркасно-обшивных конструкций

Цель: способствовать формированию знаний об организации рабочего места при подготовке поверхностей к монтажу каркасно-обшивных конструкций. Задание: составить схему организации рабочего места при подготовке поверхностей к монтажу каркасно-обшивных конструкций.

Алгоритм выполнения:

• прочитать параграф:

• проанализировать организацию рабочего места к монтажу каркасно-обшивных конструкций: определить место для звена (членов бригады). Необходимое оборудование, инструменты, приспособления инвентарь для подготовки поверхности к монтажу каркасно-обшивных конструкций:

• нарисовать схему организации рабочего места для монтажа каркасно-обшивных конструкций:

• указать необходимые инструменты, приспособления, инвентарь для подготовки поверхности к монтажу каркасно-обшивных конструкций, положение членов звена схематично (численное обозначение проводить по часовой стрелке). Форма представления задания: письменная

Критерии самооценки выполненной работы: аккуратность рисунка схемы, обозначение произведены по часовой стрелке аргументировано показано расположение членов звена, учтены необходимые инструменты, приспособления, инвентарь для подготовки поверхности к монтажу каркасно-обшивных конструкций.

Виды контроля качества выполненной работы: текущий, письменный.

Время выполнения задания; 1 ч.

Практическая работа защищается в форме устного ответа
