Практическое задание 4. Разработка инструкционно-технологических карт для выполнения работ по подготовке поверхностей

Цель: способствовать формированию знаний о последовательности технологических операций, способах их выполнения, под- бору необходимого инструмента, приспособлений, инвентаря для выполнения работ по подготовке поверхностей к монтажу каркасно-обшивных конструкций.

Задание: составить инструкционно-технологическую карту для выполнения работ по подготовке кирпичной поверхности к монтажу каркасно-обшивных конструкций. Заполнить табл. 1.3.

Алгоритм выполнения:

- прочитать параграф;

- проанализировать этапы работ по подготовке поверхностей к монтажу каркасно-обшивных конструкций:

-определить последовательность работ по подготовке поверхностей к монтажу каркасно-обшивных конструкций: отобрать инструмент, приспособления, инвентарь для выполнения работ:

-описать порядок производства работ.

Форма представления задания: письменная.

Критерии самооценки выполненной работы: аргументированность предложенных этапов, их четкая продуманная последовательность; обоснованность выбора инструментов, приспособлений, инвентаря; описание технологии производства работ. Виды контроля качества выполненной работы: текущий письменный.
