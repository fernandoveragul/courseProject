Практическое задание 4. Построение схемы организации рабочего места

Цель: способствовать формированию аналитических умений, практического опыта работы с технологической документацией, знаний по организации рабочего места по монтажу подвесного потолка.

Задание:

· прочитать гл, 3;

· выбрать ручной и электроинструмент для производства работ по монтажу подвесного потолка, выделить приспособления для подъема ГСП, выбрать средство подмащивания (стремянку, двустороннюю стремянку, столик);

· нарисовать схему помещения с указанием размеров (длина, ширина, высота).

Алгоритм выполнения:

· прочитать гл. 3;

· «Типовые строительные конструкции, изделия и узлы». Серия 1.031.9-2.07 «Комплектные системы КНАУФ. Перегородки поэлементной сборки из гипсовых строительных плит на металлическом и деревянном каркасах для жилых, общественных и производственных зданий»;

· выбрать необходимый инструмент, оборудование, материалы, средства подмащивания, продумать их удобное размещение.

Форма представления задания: письменная.

Критерии самооценки выполненной работы: аргументированность представленных данных, полнота представления информации.

Виды контроля качества выполненной работы: текущий, письменный.

Время выполнения задания: 1 ч.

Список используемой литературы: «Типовые строительные конструкции, изделия и узлы». Серия 1.031.9-2.07 «Комплектные системы КНАУФ. Перегородки поэлементной сборки из гипсовых строительных плит на металлическом и деревянном каркасах для жилых, общественных и производственных зданий». Выпуск 3 «Перегородки. Рабочие чертежи».
