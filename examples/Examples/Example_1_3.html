Практическое задание 3. Разработка последовательности технологических операций для выполнения работ по подготовке поверхностей под монтаж каркасно-обшивных конструкций

Цель: способствовать формированию знаний о последовательности технологических операций для выполнения работ по подготовке поверхностей под монтаж каркасно-обшивных конструкций

Задание: составить схему последовательности технологических операций для выполнения работ по подготовке поверхностей под монтаж каркасно-обшивных конструкций (рис. 1.3).

Алгоритм выполнения:

• прочитать параграф:

• проанализировать этапы работ по подготовке поверхностей к монтажу каркасно-обшивных конструкций; рассмотреть последовательность работ по подготовке поверхностей к монтажу каркасно-обшивных конструкций.

Форма представления задания: письменная.

Критерии самооценки выполненной работы: аргументированность выбранных этапов, их четкая продуманная последовательность, обоснованность выбора инструментов, приспособлений, инвентаря: полнота представления информации.

Виды контроля качества выполненной работы: текущий, письменный.

Время выполнения задания: 1 ч. Практическая работа защищается в форме устного ответа.
